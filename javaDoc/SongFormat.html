First Line:

Title,Time Sig,Time Sig,Tempo,Flats:Flats,Sharps:Sharps



Notes:


  • NOTE.DUR = NOTE for duration 1/DUR
  • e.g. A3.1 = A in Octave 3 for a whole beat
  • NOTE*DUR = NOTE for duration DUR
  • e.g. C4#*1.75 = C sharp in Octave 4 for 1 and 3/4 whole notes
  • Append '~' to indicate a trill
  • Place a '$' after the note name to cancel a sharp/flat
  • _ = A rest


Methods:


  • VOL:nvol = Changes dynamic to nvol
  • e.g. VOL:FF = Changes dynamic to Fortissimo
  • CRESC:nvol:num = Change dynamic to nvol over num notes
  • e.g. CRESC:P:5 = Change dynamic to Piano in 5 notes
  • MARK:txt = Create a mark called txt at the location
  • GOTO:txt = Go to the mark called txt in the music and delete it
  • INST:ninst = Change instrument to ninst
